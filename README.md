# Fyze Website

Landing page institucional da Fyze, construída com React, Vite e Tailwind CSS.

## Requisitos

- Node.js

## Desenvolvimento local

1. Instale as dependências:
   `npm install`
2. Inicie o servidor local:
   `npm run dev`

Para usar uma porta específica:

`npm run dev -- --host 0.0.0.0 --port 3001`

## Build de produção

Para gerar o build:

`npm run build`

## GitHub Pages
